# GitHub Stars Manager - 快速参考

## 🚀 3 分钟快速上手

### 1️⃣ 初次设置（30秒）
```
GitHub设置 → Personal access tokens → Generate new token (classic)
权限：勾选 repo ✅
复制令牌 → Obsidian设置 → GitHub Stars Manager → 粘贴令牌
```

### 2️⃣ 开始使用（1分钟）
```
左侧面板 🌟 图标 → 点击"同步"按钮 → 等待加载完成
```

### 3️⃣ 基本操作（30秒）
```
🔍 搜索：输入关键词实时筛选
🏷️ 标签：点击标签筛选，点击"清除全部"重置
📊 排序：点击工具栏图标切换排序方式
✏️ 编辑：点击卡片"编辑"按钮添加标签和笔记
```

## ⚡ 常用操作快捷指南

### 搜索与筛选
| 操作 | 方法 | 提示 |
|------|------|------|
| 搜索仓库 | 在搜索框输入关键词 | 支持名称、描述、语言搜索 |
| 清除搜索 | 悬停搜索框显示×按钮 | 或直接清空输入框 |
| 标签筛选 | 点击任意标签 | 可多选，再次点击取消选择 |
| 清除筛选 | 点击"清除全部"按钮 | 移除所有标签筛选条件 |

### 排序方式
| 图标 | 排序方式 | 说明 |
|------|----------|------|
| 📅 | 最近添加 | 新加星标的在前 |
| ⭐ | Star数量 | 热门项目在前 |
| 🍴 | Fork数量 | 活跃项目在前 |
| 🔤 | 字母顺序 | A-Z排列 |

### 标签管理
```
添加标签：编辑 → 标签字段 → 用逗号分隔多个标签
推荐分类：
  - 技术：frontend, backend, mobile, ai/ml
  - 用途：tool, learning, work, personal  
  - 状态：active, archived, todo
```

### 主题切换
```
默认主题：简洁卡片，与Obsidian原生主题一致
液态玻璃：iOS风格毛玻璃效果，彩色渐变背景
切换方法：点击工具栏右侧主题按钮
```

## 🛠️ 高级功能

### 多账号管理
```
添加账号：工具栏 → 账号下拉 → 添加账号
管理账号：启用/禁用开关控制同步
账号信息：显示头像、用户名、同步时间
```

### 导出功能
```
支持格式：JSON, CSV, Markdown, 纯文本
操作步骤：导出按钮 → 选择仓库 → 选择格式 → 确认导出
导出内容：完整仓库信息 + 个人标签和笔记
```

### 笔记功能
```
仓库笔记：记录个人体验、重要功能、学习计划
关联笔记：建立仓库与Obsidian笔记的双向链接
支持格式：Markdown文本格式
```

## 🔧 常见问题解决

### 同步问题
```
问题：令牌无效
解决：检查令牌是否完整、权限是否足够、是否过期

问题：同步失败  
解决：检查网络连接、API限制、令牌权限
```

### 界面问题
```
问题：搜索结果不准确
解决：搜索范围包括名称、描述、语言、标签

问题：卡片显示不全
解决：调整窗口大小、切换主题、更新插件
```

### 数据问题
```
问题：标签丢失
解决：从备份恢复、重新添加、导出数据备份

问题：仓库数量不对
解决：检查私有仓库权限、多账号设置、令牌权限
```
